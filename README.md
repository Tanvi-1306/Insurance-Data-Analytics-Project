# Insurance Data Analytics Project

## Project Overview
The Insurance Data Analytics Project is an end-to-end data analysis project that uses a single insurance dataset to perform analysis across Excel, SQL, and Tableau. The project focuses on extracting meaningful business insights related to insurance policies, customers, premiums, and claims.

This project is designed to reflect real-world insurance analytics use cases and demonstrate practical data analyst and MIS skills using multiple tools on the same data source.

## Objectives
- Analyze insurance policy and claims data
- Identify trends in premiums and claims
- Understand customer behavior and policy performance
- Build dashboards for data-driven decision-making

## Dataset Description
The dataset includes:
- Customer demographics
- Policy details
- Premium and claim amounts
- Claim status and policy status

## Tools Used
- Excel: Data cleaning, pivot tables, dashboards
- SQL: Data extraction, joins, aggregations, analysis queries
- Tableau: Interactive dashboards and visual analytics

## Key Insights
- Claim trends across policy types
- High-risk customer identification
- Premium contribution by policy category
- Policy performance and renewal patterns
